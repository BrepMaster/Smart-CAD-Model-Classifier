## Smart CAD Model Classifier

一个功能强大的 CAD 模型文件智能分类工具，基于 PythonOCC 和 PyQt5 开发，支持多种分类策略和文件格式。

通过网盘分享的文件：Smart CAD Model Classifier.exe
链接: https://pan.baidu.com/s/1j3ADSm9mvg0lH2NOBrhTrw?pwd=fwk5 提取码: fwk5 

## 功能特点

- **多种分类策略**：
  - 按颜色分类（提取模型主色）
  - 按文件大小分类（十档精细划分）
  - 按文件格式分类
  - 组合策略（先格式再颜色）

- **支持多种文件格式**：
  - .step / .stp（STEP 格式）
  - .stl（STL 格式）
  - .obj（OBJ 格式）
  - .3mf（3MF 格式）

- **用户友好界面**：
  - 支持拖拽文件夹到输入框
  - 实时进度显示
  - 详细的运行日志
  - 日志导出功能
  - 快捷帮助与详细说明
  - 美观的界面设计

- **灵活的操作模式**：
  - 复制文件（保留原文件）
  - 移动文件（移除原文件）
